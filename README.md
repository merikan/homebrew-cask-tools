# Homebrew Cask tools

This is my Homebrew Cask Tap where I'm brewing some nice tools for easy installation with [Homebrew](https://brew.sh/)

### Install a Cask
```bash
# tap this tap
brew tap merikan/cask-tools
# install
brew cask install <cask>
```
